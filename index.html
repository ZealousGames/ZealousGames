ZealousGames is a studio designated to creating/sharing games, music, coding, and films, started in 2024. The games are made using JavaScript, music is made using FL Studio, and the videos are filmed using Quicktime and edited using CapCut. The main focus of ZealousGames is to create entertainment through games and other sources.
